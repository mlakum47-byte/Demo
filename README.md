# Hello World! Demo Project

This is a simple demonstration project for a web development assignment. The goal is to display a "Hello World!" message on a webpage.

This project consists of a single `index.html` file.

## How to Run

To view the webpage, follow these steps:

1.  Locate the `index.html` file in the project directory.
2.  Open the `index.html` file in any modern web browser (e.g., Google Chrome, Firefox, Microsoft Edge).
3.  The browser will render the page and display the "Hello World!" message.
